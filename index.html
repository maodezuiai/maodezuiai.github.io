Unity常用命令

发表于 2020-10-14 更新于 2020-10-26 分类于 笔记
  • 切记，normal的转到世界空间是vertex的逆转置矩阵。
  • -worldLightDir：入射光线要反向。

  • WorldSpaceLightDir、UnityWorldSpaceLightDir、ObjSpaceLightDir稍微复杂点，帮助处理了不同种类光源问题。

  • reflect(i,n):计算反射。

  • pow(n,r)：n的r次方。
  • saturate：将数值映射到0到1。

Unityxxx为Unity5新增的内置函数。

  • UnityObjectToClipPos：模型、世界、投影矩阵。

  • UNITY_LIGHTMODEL_AMBIENT.xyz：取得环境光照。

  • _WorldSpaceCameraPos.xyz：计算相机位置。

  • ObjSpaceViewDir(float4 v)仅可用于前向渲染中。输入一个模型空间中的顶点位置，返回模型空间中从该点到摄像机的观察方向。
  • _WorldSpaceLightPos0.xyz:光源方向（位置）。

  • ObjSpaceLightDir(float4 v)仅可用于前向渲染中。输入一个模型空间中的顶点位置，返回模型空间中从该点到光源的光照方向。

  • _LightColor0.rgb：得到该Pass的光源颜色和强度信息。

  • unity_WorldToObject：转换空间坐标，注意左乘右乘的区别。

  • sampler2D 采样2D纹理。

  • tangent后跟.xyz 不知道原因

Unity Shader

发表于 2020-08-31 更新于 2020-10-29 分类于 图形学

Unity Shader基础

概述

UnityShader！= Shader

  • 传统Shader中可以编写特定的类型的Shader如顶点/片元着色器（分开编写）
    Unity Shader可以包含在同一个文件中
  • 传统Shader无法设置渲染设置，需要其它代码设置
    Unity Shader则可以
  • 传统Shader需要小心处理输入输出关心
    Unity Shader则只需声明属性即可，并且提供了模型自身属性的直接访问方式，不需要另外编码传递给着色器
  • 传统Shader更具有灵活性
    Unity Shader因为高度封装，对有些着色器的支持就差点
